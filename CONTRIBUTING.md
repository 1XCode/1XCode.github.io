# Contributing to programming-zone.github.io

:+1::tada: First off, thanks for taking the time to contribute! :tada::+1:

The following is a set of guidelines for contributing to Programming Zone, which are hosted in the [Programming Zone Organization](https://github.com/programming-zone) on GitHub. These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.
